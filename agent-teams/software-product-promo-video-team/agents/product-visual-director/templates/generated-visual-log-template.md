# Generated Visual Log

## Logging Rule

- Record one entry for every generated or edited promo visual.
- Generated and edited visuals may be first-class promo assets, but must not impersonate unsupported product UI, data, features, proof, pricing, or results.
- Image-generation calls may run in parallel or batches when the active runtime supports high-throughput generation.
