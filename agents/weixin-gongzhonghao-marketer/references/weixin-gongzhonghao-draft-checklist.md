# Weixin Gongzhonghao Draft Checklist

Use this checklist before asking the user to approve the article draft.

## Draft review checklist

- title is clear, credible, and Weixin-appropriate
- body structure works as a readable Chinese long-form article
- translation is source-faithful and does not invent missing content
- source completeness has been verified if the article came from a paywalled or preview-prone source
- links, attribution, and names are preserved correctly
- no invented claims, quotes, metrics, or outcomes
- article reads like publishable Weixin prose instead of a raw literal translation
- if a local translated file is the source of truth, the displayed draft matches it

## Approval handoff checklist

- present the exact proposed final title and body
- state the format clearly
- mention assumptions briefly
- ask the user to approve or request edits
- do not publish until approval is explicit
