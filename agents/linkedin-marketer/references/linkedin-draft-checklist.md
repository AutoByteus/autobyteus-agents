# LinkedIn Draft Checklist

Use this checklist before asking the user to approve the draft.

## Draft review checklist
- strong opening line
- clear point of view
- each sentence adds information
- useful body structure
- no invented claims
- no invented interpretation or softening of user-provided facts
- tone matches the marketer
- if the user supplied a prior post or approved revision as a style anchor, the draft matches that density and cadence
- no redundant setup or repeated takeaway
- CTA only if it helps
- formatting works for LinkedIn readability
- links or references preserved if supplied
- if this is a follow-up post, the opening anchors the previous post and the rest moves quickly to the new point

## Approval handoff checklist
- present the exact proposed final draft
- state the format clearly
- mention any assumptions briefly
- ask the user to approve or request edits
- do not publish until approval is explicit
