# LinkedIn Writing Style Reference

Use this reference when the user wants tighter writing, a fact-dense post, or a follow-up to an earlier LinkedIn post.

## Core Principle

Write short, information-dense LinkedIn copy.

Each sentence should do at least one of these:

- state a fact the user provided
- state an observation the user explicitly made
- state a conclusion clearly supported by the provided facts

If a sentence does not carry real information, cut it.

## Follow-Up Post Pattern

When the user says the new post is a follow-up:

1. Open by anchoring the post to the earlier one.
2. State the earlier setup in one or two sentences.
3. State the limitation or change directly.
4. Give concrete examples, tasks, or evidence.
5. End with one clear takeaway.

Do not re-explain background the audience already has unless it is needed for the new point.

## Sentence-Level Rules

- Prefer short declarative sentences.
- Keep connective phrases minimal.
- Use numbered lists only for discrete tasks, examples, or comparisons.
- Keep paragraphs short.
- Use the user's technical nouns and exact framing where possible.

## Avoid

- novel-like transitions
- scene-setting that does not add information
- generic motivation
- filler emphasis
- repeating the same point in different words
- explanatory softeners the user did not ask for
- invented interpretation of the user's facts

## Source-Faithful Phrasing

If the user supplies the framing, keep it.

Examples:

- If the user says a context window is `too small`, keep `too small`.
- Do not replace it with softer commentary like `may sound large` or `gets tight surprisingly fast`.
- Do not add emotional commentary unless the user expressed it.

## Revision Rule

When the user asks for denser writing:

1. remove explanation before removing facts
2. remove repeated framing before removing examples
3. keep the main takeaway explicit
