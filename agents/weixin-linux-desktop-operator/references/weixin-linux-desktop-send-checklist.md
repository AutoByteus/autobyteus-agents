# Weixin Linux Desktop Send Checklist

Use this checklist before any live send.

## Pre-send checklist

- exact target group name is confirmed
- exact final message text or final link is confirmed, or exact local image file path is confirmed
- correct desktop Weixin window is active
- search results screenshot is captured
- target row is verified from the screenshot
- opened chat title screenshot is captured
- opened chat title matches the intended group
- operator is ready to capture a post-send screenshot

## Safety handoff checklist

- do not send until the chat title is verified
- do not send if the target row is ambiguous
- send only once
- capture a post-send screenshot immediately after send
